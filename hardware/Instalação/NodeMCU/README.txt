Esses arquivos são apenas para projeto esquemático e PCB.

Como instalar:

Etapa 1: cole todos esses arquivos USERDVC.LIB, USERDVC.IDX e
USERPKG.LIB, USERPKG.IDX para a pasta Library do Proteus

e

ESP8277.STP em para pasta MCAD do Software Proteus.

Etapa 2: Se você estiver usando o software Proteus 8, a pasta Library e a pasta MCAD estarão dentro da pasta de dados (Proteus 8 ProfessionalDataLIBRARY) OU ProgramData (C: ProgramDataLabcenter ElectronicsProteus 8 ProfessionalLIBRARY e MCAD Folder)

Etapa 3: Agora abra o software Proteus (reinicie-o se já estiver aberto) e clique na guia Adicionar componente e procure NodeMCU.
